# Claude Code Study

> 基于 Claude Code 源码的原理学习网站 — **Tencent AI pkushinnxu**

深入拆解一个工业级 AI Agent 系统的架构设计，涵盖：

- Agent Loop / Tool 与沙箱 / 子 Agent 设计
- 任务图 / System Prompt / 上下文压缩
- Memory 系统 / 后台任务 / Skill & MCP
- 桌宠系统 / Team/Swarm / Ink 渲染引擎

## 本地开发

```bash
npm install
npm run dev
```

## 构建部署

```bash
npm run build   # 输出到 doc_build/
```
